## DayTrippers
 Day by Day your kinda way 

***

### Synopsis
 A MERN Stack Web app which suggests interary for a day of fun and food where ever you are. 

***

#### Technologies Used

* Javacript
* HTML
* Bootstrap
* CSS
* React
* MongoDB
* google/maps
* NPM Packages
	* axios
	* body-parser
	* express
	* google-places
	* mongoose
	* morgan
	* react-dom
	* babel-core
	* babel-loader
	* babel-preset-es2015
	* babel-preset-react
	* webpack
